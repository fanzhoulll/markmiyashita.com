Projects

A set of projects that I have recently built. A full profile can be found on my Github account.
Sound of Dub (intense-journey-6792.herokuapp.com)
A music app that I built at the Hackers@Berkeley and IEEE hackathon. Using the API, I grab songs from Soundcloud and allow users to form their own playlists or listen in a radio-style fashion.
Tim (https://timtodo.herokuapp.com/)
Winner of the Hilfinger award at the Fall 2012 HackJam. A vim-style web todo app.
Life Logger
I worked on a team of four at the 2012 Battle of the Bay Facebook Hackathon. We built an event logger for your computer that logs mouse clicks, application switches, and keystrokes. Then we analyzed and displayed the data using jquery graphs.
Pets Alliance
A massively multiplayer online role-playing game/side project which I started in June of 2012. I have built the game from scratch using Ruby on Rails.
markmiyashita.com
My personal website that I have developed from scratch. It is built using Jekyll, a static blog generator.
imbyter.com (no longer active)
Technology site that attracted over 40,000 unique users and its success eventually led to my job at BinaryAge.
bashprofile.com
Another website that I started as a technology news site. Currently under major development.
